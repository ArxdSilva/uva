- MODELO LÓGICO

- Descreve os componentes do modelo conceitual de dados, aproximando-o do ambiente computacional, onde este sera trabalhado.

    - Modelo de Redes
        Dados sao representados por colecao de registros e os relacionamentos entre os dados sao representados por meio de links;
    - Modelo Hierarquico
        Mesma estrutura de rede, mas a org. de registros é diferente. 
    - Modelo relacional
        representado atraves de tabelas.
    - Modelo relacional objeto
        extensao do relacional, q inclui orientacao a objeto e permite o tratamento de dados complexos.

    - MODELO RELACIONAL
        - integridade referencial (ler sobre)
        - chave candidata (unica, tende a se tornar a primaria - simples ou composta)
        - chave primaria
            escolhida entre as candidatas, uma tupla (linha da tabela)
            schema da tabela: linha dos atributos da tabela
        - chave estrangeira
